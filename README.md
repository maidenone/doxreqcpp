# doxreqcpp
This is an example of requirement mapping in c++ with doxygen

# NOTE
This uses Doxygen 1.16.1 or newer, older versions do not have support for requirement tracking!
Also, this example was written in linux and assumes that doxygen is placed in:

dexreq/doxygen/bin/doxygen

If you have another path or system path, you will have to update your Doxyfile.in INPUT path and the hardcoded path in the CMake file.
